# Big Five (OCEAN) Personality Traits

Holomime personality specs use the Big Five model with 20 sub-facets. Each dimension scores 0.0 to 1.0.

## Dimensions

### Openness (0.0 = conventional, 1.0 = exploratory)
How willing the agent is to explore new ideas, be creative, and deviate from established patterns.
- **Imagination**: creative vs practical
- **Curiosity**: inquisitive vs focused
- **Adventurousness**: experimental vs cautious
- **Aesthetic sensitivity**: expressive vs functional

### Conscientiousness (0.0 = flexible, 1.0 = disciplined)
How structured, organized, and detail-oriented the agent is.
- **Self-discipline**: persistent vs adaptable
- **Orderliness**: systematic vs spontaneous
- **Dutifulness**: rule-following vs pragmatic
- **Achievement-striving**: ambitious vs relaxed

### Extraversion (0.0 = reserved, 1.0 = outgoing)
How energetically the agent engages, initiates, and drives conversation.
- **Assertiveness**: directive vs deferential
- **Enthusiasm**: animated vs measured
- **Sociability**: engaging vs independent
- **Activity level**: proactive vs reactive

### Agreeableness (0.0 = challenging, 1.0 = accommodating)
How much the agent prioritizes harmony, empathy, and cooperation.
- **Compassion**: empathetic vs objective
- **Cooperativeness**: collaborative vs independent
- **Trust**: trusting vs skeptical
- **Modesty**: humble vs confident

### Emotional Stability (0.0 = reactive, 1.0 = steady)
How consistently the agent maintains composure under pressure.
- **Calm**: composed vs excitable
- **Resilience**: recovers quickly vs dwells
- **Confidence**: self-assured vs anxious
- **Stress tolerance**: handles pressure vs avoids it

## How Scores Map to Behavior

| Score Range | Interpretation |
|-------------|---------------|
| 0.0 - 0.2 | Very low — opposite end of the spectrum |
| 0.2 - 0.4 | Low — leans toward the left pole |
| 0.4 - 0.6 | Moderate — balanced, situational |
| 0.6 - 0.8 | High — leans toward the right pole |
| 0.8 - 1.0 | Very high — strong expression of the trait |
